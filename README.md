# Mavs Jewelry - Joyas Artesanales en Plata

Sitio web oficial de **Mavs Jewelry**, un emprendimiento de joyería artesanal en plata con piedras semipreciosas, mostacilla calibrada y alambrismo. Hechas a mano por María Angélica, orfebre de oficio.

## Descripción del Proyecto

Página web informativa y de exhibición para el emprendimiento Mavs Jewelry. Incluye:

- **Slider principal** con catálogo de productos (plata, mostacilla, alambrismo)
- **Sección Quiénes Somos** con la historia del emprendimiento
- **Galería de productos** con carrusel interactivo
- **Testimonios** de clientes
- **Misión y Visión** del emprendimiento
- **Galería de imágenes** con lightbox
- **Redes sociales** flotantes con acceso directo a Instagram, TikTok, Facebook y WhatsApp
- **Footer** con enlaces y datos de contacto

## Tecnologías Utilizadas

- HTML5
- CSS3 (con variables personalizadas)
- JavaScript (lightbox, navegación scroll)
- Bootstrap 4
- Revolution Slider
- Owl Carousel
- Font Awesome 6
- Google Fonts (Cormorant Garamond + Montserrat)
